# Raytracing CSG Renderer

## Getting Started

```sh
sh scripts/usage.sh
```

## Usage

1. Write the scene in JSON, YAML, or TOML format (jsonc, json5, and hjson are also supported).
2. Run the following command:
   ```sh
   cargo run --release [options] <path-to-scene> <path-to-output>
   ```
